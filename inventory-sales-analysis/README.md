# Inventory & Sales Data Analysis

## Objective
To analyze inventory and sales data in order to identify sales trends, top-performing products, and inventory management insights that support business decision-making.

## Tools Used
- Microsoft Excel
- Python (Pandas, Matplotlib)

## Key Tasks
- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Sales trend analysis
- Inventory level evaluation

## Expected Insights
- Best-selling products
- Low-performing inventory items
- Monthly sales trends
- Inventory optimization recommendations
